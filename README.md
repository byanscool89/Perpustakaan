        <h2>Cara Memulai</h2>
        <ol>
            <li>Clone repositori ini ke komputer Anda.</li>
            <li>Buka terminal dan navigasi ke direktori proyek.</li>
            <li>Install dependensi terlebih dahulu.</li>
        </ol>
        <pre>npm install axios</pre>
        <pre>npm install react-router-dom localforage match-sorter sort-by</pre>
        <pre>npm i swiper</pre>
        <pre>npm i -D daisyui@latest</pre>
        
        <h3>Jalankan perintah:</h3>
        <pre>npm run build</pre>
        <p>Digunakan untuk membuat build produksi dari proyek. Ini akan menghasilkan file-file yang siap untuk digunakan di lingkungan produksi.</p>
        
        <pre>npm run dev</pre>
        <p>Perintah ini digunakan untuk memulai lingkungan pengembangan. Ini akan menjalankan server pengembangan dan memungkinkan Anda untuk menguji proyek Anda secara lokal.</p>
        
        <pre>npm run preview</pre>
        <p>Perintah ini digunakan untuk membuat lingkungan review. Ini akan menjalankan proyek dalam mode tampilan pratinjau sehingga Anda dapat melihat bagaimana proyek akan terlihat di lingkungan produksi.</p>
